aQuery - Write less.

What is it?

aQuery is a super-new fast and concise JavaScript Library that simplifies HTML document traversing, event handling, animating, Ajax interactions, and barnacle scrubbing for rapid web development. aQuery is designed to change the way you view the world around you.
